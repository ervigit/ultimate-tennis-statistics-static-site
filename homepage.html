Ultimate Tennis Statistics

This is the most complete app to collect statistics for tennis match.

In this era of data, if you want to improve your tennis skills, you need data. This app is exactly for that. It allows to note, for each point in the match, the way each point ended and you can afterwards watch a whole batch of statistics.

Features

Here are the main features:

  • easily enter the way each point has ended
    • ace, service winner, service return
    • winner, unforced error, forced error
      • forehand, backhand
      • topspin, slice, volley, smash, drop, passing, swing volley
    • note the attitude and routine of each player
    • you can even write a note on a point
  • score is then computed automatically
  • jump to a specified score if you missed some part of the match
  • share the score as the match goes (by sending text messages)
    • at each point
    • at the end of each game
    • at the end of each set
  • look the match statistics
    • for the whole match
    • for each set
    • they can be easily shared as an image
  • show a the graph of the match to look for trends
    • each point of the graph is button to look exactly what happened at that point
    • it can be easily shared as an image
  • share the match
    • so another user can open the match in its own device and look for the statistics or graphs
  • all match formats are supported
  • all the matches are kept so you can review your tennis statistics afterwards

The Unlimited Library in-app purchase allows to store an unlimited number of matches win your library.

The Advanced in-app purchase allows to qualify each strokes more precisely:

  • winner, forced error
    • deep, short
    • crosscourt, parallel, inside-in, inside-out
  • 2nd stroke after service or after return
